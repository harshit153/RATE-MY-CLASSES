# RATE-MY-CLASSES
A forum application using MERN stack which allows users to rate and post reviews about classes.
<br></br>
Added functionality for sorting reviews by time posted, semester taken, and no. of likes & searching based on class name, college, professor. Implemented user authentication using JSON web token to secure required APIs for adding and deleting own reviews
